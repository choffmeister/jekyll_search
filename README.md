## Installation

Add this line to your application's Gemfile:

```ruby
source 'https://rubygems.org'

group :jekyll_plugins do
  gem 'jekyll_search'
end
```

or for bleeding edge:

```ruby
group :jekyll_plugins do
  gem 'jekyll_search', :git => 'https://github.com/choffmeister/jekyll_search.git', :branch => 'develop'
end
```

And then execute:

    $ bundle
